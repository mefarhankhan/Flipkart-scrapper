# Flipkart Scraper

A lightweight and efficient Flipkart scraper built to extract product details such as title, price, ratings, reviews, discounts, specifications, seller information, and availability directly from Flipkart listings and product pages.

## Features

* Scrape product title, price, and ratings
* Extract product descriptions and specifications
* Collect reviews and seller details
* Supports category and search result scraping
* Fast and easy-to-use implementation
* Clean and structured output (JSON/CSV)
* Customizable scraping logic

## Tech Stack

* Python
* Playwright / Selenium
* Requests
* Pandas

## Use Cases

* E-commerce price tracking
* Product research and analysis
* Competitor monitoring
* Market trend analysis
* Data collection for machine learning projects

## Installation

```bash
git clone https://github.com/yourusername/flipkart-scraper.git
cd flipkart-scraper
pip install -r requirements.txt
```

## Usage

```bash
python scraper.py
```

## Output

The scraper exports collected data into:

* JSON format
* CSV format (Mostly)
